About computer

computer is an electronic data processing device.It can read, write, compute and compel stored and process large volume of data.

Data processing cycle in computer:

  • Input: Data is collected and enter into the computer.
  • Storage: Whatever is entered is stored in its main memory.When the power is switched off, the memory will be automatically lost.
  • Processing: It is the sequence of action performed on the data to get desired results.
  • Output: Stored results are taken out of the main memory.

Components of Computer

  • CPU(Central Processing Unit)
  • Input Devices(Mouse, Keyboard, etc)
  • Output device(Monitor, Speaker, etc)

Functions

  • Maths Functions
  • Logical Functions
  • Statistical Functions
  • Text Functions
  • Date & Time Functions
